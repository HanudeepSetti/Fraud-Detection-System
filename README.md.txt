# JP Morgan Fraud Detection

A machine learning fraud detection web app built using Streamlit.

## Features
- Fraud prediction
- Fraud probability score
- XGBoost model
- Streamlit web interface

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
